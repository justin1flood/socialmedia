WELCOME TO MY WEBSITE!!!


What is this website?

This website was built to cover 5 social media apps. Each app will be explained what is it, how it could be used for teaching and learning, how to register for the app, any cautions or warnings about the app and finally videos/pictures of what the application looks like and how to use it.


What else can you expect to see?

There are more pages than just the social media applications. There is also:

A contact page including a feedback form where you can give the owner feedback on what they can improve.

A contents page that has a table and in that table has a description on what each page is about.

There is a further resources page with a logo of all the five social media applications covered on this website and when you click the logo it will bring you to the offical website of that app where you can get more info on that specific application.

There is also an About page. On this page there will be a sitemap of this website and 2 sentences on what this website is about.
